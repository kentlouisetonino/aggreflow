### Development
> - Bash scripts.
```bash
# Build the app.
chmod +x build
./build

# Run the app.
chmod +x app
./app
```

> - Helpful guides for module management.
```bash
# Installing package.
go get <github/*>

# Make vendored copy of dependencies.
go mod vendor

# Add missing and remove unused modules.
go mod tidy
```
